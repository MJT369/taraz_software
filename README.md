# TarazSoft

## Description
TarazSoft is a comprehensive software tool for text editing, translation, and book searching on the internet. It integrates various libraries and APIs to provide a seamless user experience.

## Features
- Text editing
- Translation using multiple translation services
- Book searching on the internet
- Support for multiple languages
- User-friendly interface with PyQt5 and Tkinter

## Installation
To install TarazSoft, follow these steps:

1. Clone the repository:
    ```sh
    git clone https://github.com/MJT369/taraz_software/
    ```
2. Navigate to the project directory:
    ```sh
    cd tarazsoft
    ```
3. Install the required dependencies:
    ```sh
    pip install ...
    ```
4. Install the package:
    ```sh
    pip install .
    ```
